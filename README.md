# Ristoro Alpino Monte Basso

A website for **Ristoro Alpino Monte Basso**, a traditional alpine restaurant nestled at 1,500 m altitude in Casargo (LC), Valsassina. Serving authentic mountain cuisine since 1970.

![Screenshot](screenshot.png)

## Live Demo

[https://safaberk334.github.io/monte-basso/](https://safaberk334.github.io/monte-basso/)

## Features

- Responsive design optimized for all devices
- Full restaurant menu with pricing
- Photo gallery
- Events calendar
- Online reservation form
- Vacation rental listings
- Family history section
